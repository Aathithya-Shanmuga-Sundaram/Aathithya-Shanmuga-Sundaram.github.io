Private WordPress Site RCE

Educational Institution

Discovered high severity Remote Code Execution vulnerability via outdated plugin in the institution's WordPress site, which could allow attackers to execute arbitrary code on the server.

Found missing security headers and input validation in the institution's feedback system, increasing the risk of SQL injection attacks that could compromise the database.

Medium Severity SQL Injection Input Validation

Resolution

Developed and provided a security patch that implemented proper input validation, parameterized queries, and appropriate security headers to prevent SQL injection attacks.
